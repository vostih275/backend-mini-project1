# Backend Mini Project

## Description
Group backend project built with Node.js and Express using JSON file storage and EJS templates.

## Features
- View all blog posts
- View single post by ID
- Server-side rendering with EJS

## Technologies
- Node.js
- Express
- EJS
- File System (fs)
## How to Run
1. Clone the repository
2. Run `npm install`
3. Start the server using `node src/app.js`
4. Visit `http://localhost:3000`
